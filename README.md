# AniSubio

Русские фансаб-субтитры для аниме в Stremio.

AniSubio — живой Stremio-аддон, который добавляет русские аниме-субтитры как
обычный выбираемый источник в плеере. База пополняется фоново: аддон
устанавливается один раз, а новые субтитры появляются без переустановки.

## Установка

[Установить AniSubio в Stremio](https://82-146-41-23.sslip.io/anisubio/)

После установки AniSubio появляется среди доступных источников субтитров
в плеере Stremio. База пополняется автоматически, поэтому переустанавливать
аддон для получения новых субтитров не требуется.

Если Stremio не открыл установку автоматически, добавьте manifest URL вручную:

```text
https://82-146-41-23.sslip.io/anisubio/v2/manifest.json
```

## Зачем нужен AniSubio

Stremio давно зарекомендовал себя как одна из самых универсальных платформ
для просмотра фильмов, сериалов и аниме. Для классического кино и западных
сериалов русскоязычный сегмент уже достаточно хорошо обустроен: работают
торрент-парсеры и различные агрегаторы. Однако для ценителей русского
аниме-фансаба Stremio до сих пор практически не был адаптирован.

### В чём проблема

**Опустевшие глобальные базы.** Стандартные решения вроде OpenSubtitles
хорошо справляются с голливудскими релизами, но для аниме там критически мало
качественных русских субтитров.

**Изоляция фансаб-наследия.** Огромное количество качественных авторских
переводов с выверенными таймингами и оформлением годами оставалось в архивах
нишевых сайтов и старых сообществ, отдельно от современных медиаплатформ.

**Привычка к компромиссам.** Многие зрители продолжают смотреть аниме на
онлайн-сайтах с пережатым видео, рекламой и вшитыми субтитрами просто потому,
что удобной альтернативы долгое время не существовало.

AniSubio создаётся как мост между трудом русских переводчиков и современным
способом просмотра аниме. Для первой версии проекта был проиндексирован архив
**fansubs.ru**, а найденные русские субтитры постепенно становятся доступны
пользователям аддона.

AniSubio не содержит видео и не предоставляет видеопотоки. Аддон добавляет
только внешние русские субтитры для выбранного пользователем источника видео.
